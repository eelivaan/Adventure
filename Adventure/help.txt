Useful keys:

[Enter]       Submit command
[Arrow Up]    Scroll through previous commands
[Arrow Down]  Scroll through previous commands
[Esc]         Exit application
[Ctrl]        Retreat quickly


Available commands:

Movement:
go left
go right
go up
go down

Restart the game:
restart

Use a key, if you have found one,
to unlock a door:
use key
